#项目相关信息
##项目源码获取.
本项目主要用bower进行依赖管理.

###1.获取源码开发环境步骤:
* 1. clone仓库代码
```
git clone https://github.com/kirinse/kod-ui
```
结束了后.

* 2. cd 到webapp根目录下.
```
bower install
```

即可.

**tip:**
安装nodejs,yeoman,bower方式:敬请使用百度,谷歌


##1.前端页面存放位置和入口

**(1)页面存放位置:webapp/client**

其它的一律无视

**(2)入口webapp/client/index.html**

**(3)js入口webapp/client/app/app.js**
